This collection of UI redesigns focuses on improving usability, accessibility, and efficiency across various digital interfaces. By analyzing existing designs and identifying key areas for enhancement, I created quick-win solutions that seamlessly integrate into the overall user experience. ​These redesigns span different platforms and use cases, addressing unique challenges while maintaining a user-centric approach. Netflix
Browser Screens
Challenge:

The "Browse by Language" page offered a generic filter and recommendations, lacking personalization and a user-friendly design.

​

Goal: Enhance the accessibility of multilingual browsing options.​
Solution:

I’ve created a preferences option with a sentence structure that aligns with how users might think when using the feature. While this is just an idea for now, testing will help determine if it works. The example shows a user asking for a show with English audio, but it could easily extend to requests like an anime with English subtitles or an Iranian movie with Spanish audio. I’ve placed the “This Show is” tags outside so users can clearly see what they’re selecting. This will help, especially for content in languages where users might not understand the titles.

​​

TL;DR version:

  1. Adjusted the layout and provided tags to help decision making.

  2. Made the filters seem more natural/intuitive and easier to understand.

Thomas Fisher Rare Book Library
Exhibitions Page
Challenge:

The exhibitions section is fragmented across multiple pages, requiring users to navigate separately to view current, upcoming, previous, and online exhibitions. This disjointed experience makes it harder to get an overview of the library’s exhbitions at a glance.

​

Goal: Consolidate exhibition categories into one page.​
Solution:

What I’ve done:

  1. Combined the fours sub pages into a single page, reducing unnecessary navigation.

  2. Organized each section distinctly.

  3. Provided quick access to full lists of online and previous exhibitions while ensuring individual exhibitions still have dedicated pages.

  4. Designed the page to accommodate without disrupting the layout.

CBC Radio Canada
Careers Pages
Challenge:

The existing careers section was fragmented across multiple sub-pages, creating a disjointed experience. Users had to navigate several layers to find information.

​

Goal:​ Create an engaging, informative user journey for prospective employees.​
Solution:

What I've done:

  1. Consolidated career-related sub-pages into a single, structured page.

  2. Reorganized content into clearly defined sections, to reflect user needs like vacancies, company culture, and benefits.

  3. Provide an intuitive navigation system, reducing unnecessary clicks.
